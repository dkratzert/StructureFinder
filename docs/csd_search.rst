CSD Search
==========

StructureFinder is able to search for unit cells in the
`Cambridge Structural Database <https://www.ccdc.cam.ac.uk/structures/>`_ (CSD) using
the CellCheckCSD program.

As soon as CellCheckCSD is installed and its path is configured in StructureFinder,
you can search the CSD directly. The path to the CellCheckCSD executable can be
set in the application settings.

Double-click on a result row to get the detailed structure page from the CSD.
