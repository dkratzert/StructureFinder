Exporting Data
==============


CIF Export
----------

StructureFinder can export the currently selected structure as a CIF file.
Click the "Export as CIF" button to save the structure data. Note that this
CIF is reconstructed from the database and is not the original CIF file.

CIF export is also available in the web interface.


Excel Export
------------

You can export table data to an Excel (.xlsx) file. Select the rows you want
to export in the main table, then use the right-click context menu to save the
selection as an Excel spreadsheet. This is useful for creating reports or
sharing data with colleagues.
