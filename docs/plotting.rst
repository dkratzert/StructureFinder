Plotting
========

The "Plot" tab allows plotting of CIF data as scatter points or histograms.

You can select which CIF data fields to plot from the available columns in the
database. This can be useful for visualizing trends across your structure
collection, such as R1 values, unit cell volumes, or temperature distributions.

A click on a data point scrolls the main table to the respective position of the
CIF entry, making it easy to identify specific structures.
